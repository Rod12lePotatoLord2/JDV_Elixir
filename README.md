# 🕹️ Jogo da Velha - Paradigma Funcional (Elixir)

Este projeto foi desenvolvido como parte das atividades acadêmicas para demonstrar a implementação do clássico **Jogo da Velha** utilizando o **Paradigma Funcional** com a linguagem **Elixir**.

## 🚀 Como Testar e Rodar o Projeto

No VsCode, abra o terminal e digite elixir jogo_da_velha.exs [Com a linguagem instalada, claro]
